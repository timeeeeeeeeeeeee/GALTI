📖 如何收集所有用户的数据

⚠️ 问题：为什么看不到别人的数据？

数据存在每个用户自己的浏览器里，你只能看到自己的。
用 webhook.site 可以收集所有人的数据，免费，不需要任何账号。

只需两步（1分钟搞定）

第一步：获取你的专属 URL

  1. 打开 👉 webhook.site
  2. 页面会自动生成一个专属 URL，类似：
    https://webhook.site/xxxx-xxxx-xxxx
  3. 复制这个 URL

💡 网页打开后就自动有了，不需要注册任何东西

第二步：配置到你的网站

  1. 把 URL 粘贴到下面，点保存
  2. 然后在 galti_app.js 中找到 localStorage.getItem('galti_ep') 这一行附近
  3. 其实保存后就生效了！用户的数据会自动发送到 webhook.site

第三步：查看数据

  1. 保持 webhook.site 页面打开
  2. 每当有用户完成测试，页面会实时显示收到的数据
  3. 每条数据包含：时间、匹配角色、匹配度、评价、20维分数、所有答案
  4. 你可以点击每条请求查看详细 JSON 数据

✅ 就这么简单

• 用户完成测试 → 数据自动发到 webhook.site
• 你在 webhook.site 实时看到所有数据
• 完全免费，不需要任何账号
• 别人不知道你的 URL 就看不到数据

⚠️ 注意事项

• webhook.site 页面关了就收不到数据了（可以刷新重新打开）
• 免费版数据会过期，重要数据记得及时复制保存
• URL 就是你的密钥，不要泄露给别人
• 如果想要永久存储，可以用 Google Sheets 方案（需要谷歌账号）
