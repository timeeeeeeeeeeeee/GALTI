📖 如何让所有用户的数据发送到你这里

⚠️ 问题：为什么看不到别人的数据？

目前数据存在每个用户自己的浏览器里，你只能看到自己的。
要用 Google Sheets 收集所有人的数据，按下面步骤操作：

第一步：创建 Google Sheet

  1. 打开 sheets.google.com，创建新表格
  2. 第一行输入7个表头（每格一个）：
    时间戳 会话ID 匹配角色 匹配度 评价 20维分数 50题答案

第二步：添加 Apps Script

  1. 在 Google Sheet 中点击菜单 扩展 → Apps Script
  2. 删除代码编辑器里的所有内容
  3. 打开你项目中的 google_sheets_backend.js 文件
  4. 复制注释 /* ... */ 里面的代码（不包括 /* 和 */）
  5. 粘贴到 Apps Script 编辑器中
  6. 把第一行的 your_password_here 改成你的密码（记住它！）
  7. 点击 💾 保存

第三步：部署为 Web App

  1. 点击右上角 部署 → 新建部署
  2. 点击「类型」旁边的齿轮，选择 Web 应用
  3. 执行身份选 我
  4. 谁可以访问选 任何人
  5. 点击 部署，授权弹窗全部允许
  6. 复制生成的 URL（类似 https://script.google.com/macros/s/xxxx/exec）

✅ 完成后效果

• 用户完成测试 → 数据自动写入你的 Google Sheet
• 你可以随时打开 Google Sheet 查看数据
• 你也可以在本页面「云端数据」标签查看
• 别人完全看不到任何数据
• Google Sheet 可以直接导出 Excel/CSV 分析

🔒 安全保障

• Google Sheet 只有你的 Google 账号能打开
• 用户只能写入（POST），不能读取（GET）
• 读取数据需要你设置的密码
• 即使别人知道 URL 也无法查看数据
